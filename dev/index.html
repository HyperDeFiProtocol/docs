# What we are using

  • Visual Studio Code (opens new window)
  • WebStorm (opens new window) as the main IDE
  • Remix (opens new window) as the Solidity (contract) IDE
  • Node.js (opens new window)
  • Vue2 (opens new window)
  • Nuxt.js (opens new window) as the Main Framework
  • web3.js (opens new window) as the web3 support provider
  • Dexie.js (opens new window) as browser IndexedDB driver
  • TailwindCSS (opens new window) and TailwindUI (opens new window) for the styles and UI
  • HeroIcons (opens new window) as icons resource
  • Ico Generator (opens new window) for generating favicons
  • Affinity (opens new window) as designing tools: Affinity Photo (opens new window) for jpg, and Affinity Designer (opens new window) for vector.
  • GitHub Actions (opens new window) for CI
