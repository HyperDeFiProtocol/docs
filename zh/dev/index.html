# 构建

# git clone
$ git clone git@github.com:HyperDeFiProtocol/home.git

# install dependencies
$ yarn

# serve with hot reload at localhost:3000
$ yarn dev

# generate static project to `./dist`
$ yarn generate

# 我们正在使用

  • Visual Studio Code (opens new window)
  • WebStorm (opens new window) 作为主 IDE
  • Remix (opens new window) 作为 Solidity（合约）IDE
  • Node.js (opens new window)
  • Vue2 (opens new window)
  • Nuxt.js (opens new window) 作为主框架
  • web3.js (opens new window) 作为 web3 支持组件
    • Dexie.js (opens new window) 作为浏览器 IndexedDB 驱动
  • TailwindCSS (opens new window) and TailwindUI (opens new window) 样式与 UI
  • HeroIcons (opens new window) 图标资源
  • Ico Generator (opens new window) favicon 生成
  • Affinity (opens new window) 设计工具 Affinity Photo (opens new window) 输出位图、 Affinity Designer (opens new window) 输出失量
  • GitHub Actions (opens new window) 用于自动构建
